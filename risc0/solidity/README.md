# Groth16 On-Chain Verification

Integration tests for verifying Cartesi step proofs on-chain via the
RISC Zero Verifier Router on Sepolia.

## Quick Start

The easiest way to run is from the parent directory:

    make -C risc0 test         # Generates fixtures + runs all tests

To run only the Solidity tests (fixtures must already exist):

    make dep                   # Install Foundry dependencies (first time)
    make test                  # Run tests against Sepolia fork

Fixtures (`seal.bin`, `journal.bin`) are generated by the parent
`risc0/Makefile` and shared at `risc0/test/fixtures/`.

## Prerequisites

- [Foundry](https://getfoundry.sh) installed (`forge`)
- Fixtures generated by `make -C risc0 test` (requires Docker, `cartesi-risc0-cli`, `cartesi-machine.lua`)
